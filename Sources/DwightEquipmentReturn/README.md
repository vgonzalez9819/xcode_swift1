# Dwight Equipment Return

This SwiftUI app collects equipment return information from students and replaces manual sticker tracking. It requires Xcode 13 or later and runs on iOS 14+ and Mac Catalyst.

## Building

1. Open `DwightEquipmentReturn.xcodeproj` or create an Xcode project that includes the sources in `Sources/DwightEquipmentReturn/`.
2. Build and run on an iOS or Mac Catalyst target.

The form records asset tag, charger and Yondr pouch status, and other notes. Submissions are stored in memory and listed below the form. A confirmation alert appears after submission.
